# Grid Optimiser

A notebook-to-project workspace for building a **Virtual Power Plant (VPP) for Commercial Real Estate (CRE) in Kenya** and packaging it for funding programs like the **Digital Energy Challenge 2026**.

## Structure
- `startup/` — product + engineering docs and (eventually) source code
- `entrepreneurship/` — strategic business plan and growth docs
- `innovation/` — funding/challenge application pack docs
